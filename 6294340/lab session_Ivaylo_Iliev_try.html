Create an algorithm that calculates all the acceleration values and stores them in the grid. The algorithm is described visually in the last part of the lecture. At each grid point, it calculates a weighted mean of all measurements. The weights are given by an exponential function, based on the 'distance' between the grid point, and the measurement values of dv and s. To get you started, how many for-loops do you need?


For this you will need math.
Use an upsilon of 1.5m/s and a sigma of 30m.

#define the constants
upsilon = 1.5
sigma = 30

#Double for-loop so that you have access to every grid point
for i in range(0, len(s_grid)):
    s_const = s_grid[i]
    for j in range(0, len(dv_grid)):
        dv_const=dv_grid[j]

        #reset at each itteration
        sum_weight_A_i = 0
        sum_weight = 0

        #use zip to itterate at the same time over the 2
        for dv_data,s_data,A_data in zip(DV_array,S_array,A_array):

            #Calculate the weight based on the formula given for W
            w =  math.exp( -abs(dv_data - dv_const)/upsilon - abs(s_data - s_const)/sigma )

            #Calculate the 2 nominator and denominator
            sum_weight_A_i += w*A_data
            sum_weight += w 
        
        #fill-in the 0-grid from before
        a_grid[i,j] = sum_weight_A_i / sum_weight
